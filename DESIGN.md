# DESIGN.md - 花间乐章

## 气质与意象

清晨的花园，薄雾尚未散尽，阳光透过藤蔓与花瓣洒下斑驳光影。一位少女戴着耳机，坐在秋千上闭眼聆听，身旁是盛开的玫瑰与鼠尾草。音乐从黑胶唱片中缓缓流淌，与花香交织成一首无声的诗。

## 视觉策略

- **摄影/图像方向**：自然光下的花卉、植物、水彩质感。封面图使用柔和暖调处理，避免高对比度
- **图形语言**：水彩晕染效果、花瓣形容器、藤蔓曲线分割线、金色星光粒子点缀

## 配色方案

| 角色 | 色值 | 意象来源 |
|------|------|----------|
| 主色 - 鼠尾草绿 | #7D8B6E | 清晨叶片上的露珠 |
| 辅色 - 玫瑰粉 | #E8B4B8 | 花园中半开的玫瑰 |
| 强调色 - 暖金 | #C9A96E | 透过花瓣的阳光 |
| 根背景 - 米白暖底 | #FAF7F2 | 未漂白的棉麻质感 |
| 卡片背景 | rgba(255,255,255,0.85) | 薄雾中的花瓣 |
| 文字主色 | #3A3530 | 树皮上的苔藓 |
| 文字次色 | #8B7D6B | 干枯的藤蔓 |
| 阴影色相 | #7D8B6E (绿调) | 叶影婆娑 |

## 字体排版

- 标题：系统默认粗体，字号 28-32px，字间距略宽
- 正文：系统默认，15-16px
- 辅助文字：12-13px，颜色弱化

## 动效与交互

- 黑胶唱片：播放时缓慢旋转 (20s/圈)，暂停时停止
- 页面过渡：0.3s ease-out 淡入滑动
- 喜欢按钮：空心→实心粉色，带弹性缩放
- 底部迷你播放条：上滑展开为完整播放页

## 设计禁忌

- 不要使用冷色调（蓝、紫）作为主色
- 不要使用直角或小圆角（保持 20+ 的大圆角）
- 不要使用纯黑阴影（使用带绿色调的有色阴影）
- 不要使用过于饱和的颜色
- 不要使用硬边分割线（使用波浪线或留白分隔）
