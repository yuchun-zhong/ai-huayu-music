# 花语音乐 - 设计风格指南

## 气质与意象

清晨的花园里，阳光透过藤蔓洒下斑驳光影，露珠在叶片上闪烁。你戴着耳机，漫步在花丛间，每一首歌都是一朵盛开的花。

**关键词**：森系浪漫 · 水彩花园 · 温柔治愈 · 自然有机

## 配色方案

| 角色 | 色值 | 意象 |
|------|------|------|
| 根背景 | `#FAF7F2` | 清晨薄雾中的米白暖光 |
| 卡片背景 | `rgba(255,253,248,0.9)` | 花瓣上的晨露 |
| 主色 | `#7D8B6E` | 鼠尾草绿，叶片的生命力 |
| 强调色 | `#E8B4B8` | 玫瑰粉，花瓣的温柔 |
| 金色点缀 | `#C9A96E` | 阳光穿过花丛的金色光斑 |
| 主文字 | `#3A3530` | 深棕，泥土的温润 |
| 副文字 | `#8B7D6B` | 暖灰，树干的质朴 |
| 弱文字 | `#C4B8A8` | 浅灰，远山的朦胧 |

## 设计语言

- **大圆角**：卡片 20px，按钮 16px，营造花瓣般的柔和感
- **有色阴影**：使用主色调的半透明阴影，拒绝纯黑阴影
- **渐变按钮**：主色按钮使用 `#7D8B6E → #6B7A5C` 渐变
- **装饰元素**：花瓣形、叶片、藤蔓、星光粒子
- **Tab Bar**：粉绿渐变毛玻璃效果

## 设计禁忌

- 禁止冷色调（蓝、紫）
- 禁止直角/锐角设计
- 禁止纯黑阴影
- 禁止高饱和度色彩
- 禁止死白背景
